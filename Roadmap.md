# Roadmap Minecraft Holy Bot

На данный момент планируется 3 основных этапа развития программы.


## 1 Этап (текущий)

Этот этап развития кардинально меняет направление программы. 
Программа будет с открытым исходным кодом, кроссплатформенной, масштабируемой, а также с другим названием.
В программе будут удалены простые DDOS боты(боты с методами BotJoiner, Memory и т.д.), умные боты будут переименованы в стресс-тест и добавлен менеджер ботов, в котором можно будет создавать одиночных(простых) ботов с плагинами, для того, чтобы автоматизировать игровой процесс на серверах, например, АФК-рыбалка.


### 1.1 Подэтап (текущий)

Этот подэтап сфокусирован на улучшении стресс-тест ботов, а также общего состояния приложения.

- Визуально будет улучшен график с метриками стресс-тест тестирования.
- Из-за улучшенной асинхронности и переиспользования памяти будет улучшена общая производительность стресс-теста.
- В стресс-тест тестировании можно будет загружать пользовательские прокси.
- Стресс-тест боты получат возможность использовать плагины.

## 2 Этап

К моменту начала работы над этим этапом в программе будет возможность создавать плагины, но этого мало, так как библиотека McProtoNet не дает полные возможности для работы с протколом Minecraft.


- Будут убраны библиотеки McProtoNet.Protocol*. Они будут заменены на кодогенерацию, где можно будет генерировать мультиверсионные клиенты майнкрафт.
- Программа и библиотеки SDK для неё получат обновления для того, чтобы разработчики плагинов могла пользоваться новыми возможностями McProtoNet.
- В библиотеку McProtoNet будет внедрён System.IO.Pipelines, чтобы работа с протоколом была максимально быстрой в сценариях, где требуется высокая производительность, например, стресс-тест или игровой сервер.
## 3 Этап

Этот этап касается плагинов.

- Каждый плагин будет запускатся в отдельном процессе. Это нужно для того чтобы хост-программа не завершала работу аварийно, если в плагине произошла фатальная ошибка.
- Будет добавлена возможность управлять ботами через клиент Minecraft.
